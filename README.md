# SaudiCareers

منصة وظائف حكومية وشبه حكومية في المملكة العربية السعودية - مدعومة بالذكاء الاصطناعي

## المميزات

- 🔍 مطابقة ذكية للوظائف
- 📄 تحليل السيرة الذاتية بالذكاء الاصطناعي
- 🏛️ وظائف حكومية موثوقة
- 📱 تصميم متجاوب بالكامل

## التقنيات المستخدمة

- HTML5 + Tailwind CSS
- JavaScript Vanilla
- Google Fonts (Almarai, Cairo)
- Font Awesome Icons

## الصفحات

- `index.html` - الصفحة الرئيسية
- `jobs.html` - قائمة الوظائف (قريباً)
- `cv-analyzer.html` - تحليل السيرة الذاتية (قريباً)

## التطوير

```bash
# تشغيل الموقع محلياً
npx live-server

# أو ببساطة افتح index.html في المتصفح
```

## رؤية 2030

هذا المشروع يدعم رؤية المملكة العربية السعودية 2030 في تمكين المواطنين والمواطنات من الوصول لفرص عمل مميزة.
